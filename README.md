# guardiantapp

Mobile Application in React Native

## Descripción
Proyecto móvil desarrollado con React Native + TypeScript destinado a la app "Guardiant / CyberGuardian". Contiene configuración básica, navegación y conexión a Firebase.

## Tecnologías
- React Native (Template TypeScript)
- TypeScript
- Firebase
- @react-navigation/native, @react-navigation/stack
- react-native-safe-area-context, react-native-screens

## Requisitos
- Node.js (versión recomendada compatible con la versión de React Native usada)
- npm o yarn
- JDK + Android Studio para Android
- Xcode para iOS (macOS)
- react-native-cli (opcional)
